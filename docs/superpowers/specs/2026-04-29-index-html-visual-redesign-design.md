# LLM API 转换器管理页面视觉重塑设计

## 目标
将 `static/index.html` 的视觉风格从现有的「靛蓝/深灰企业风」重塑为「Claude 白天模式」的温暖极简风格，提升品质感与可读性，同时保持现有功能和布局结构不变。

## 设计原则
1. **温暖极简**：以暖米白为底，去除冷硬的蓝灰色调，营造柔和、专业的阅读体验。
2. **呼吸感**：增加间距与留白，减少视觉压迫感。
3. **层次清晰**：通过字体大小、字重和色彩对比建立明确的信息层级，而非依赖背景色块分割。
4. **精致微交互**：hover、focus、激活态均带有平滑过渡，提升交互质感。

## 配色方案

| 用途 | 色值 | 说明 |
|------|------|------|
| 页面背景 | `#faf9f7` | 暖米白，Claude 白天模式的典型底色 |
| 卡片/浮层背景 | `#ffffff` | 纯白，与页面背景形成 subtle 对比 |
| 主文字 | `#1a1a1a` | 深炭灰，接近黑但柔和 |
| 次要文字 | `#6b6b6b` | 中灰，用于描述、辅助信息 |
| 禁用/占位文字 | `#9ca3af` | 浅灰，保留现有 Tailwind slate-400 |
| 强调色 | `#635bff` | 深紫，用于主按钮、激活态、核心标签 |
| 强调色 hover | `#554ce6` | 比主强调色略深 |
| 边框/分割线 | `#e8e6e3` | 极浅暖灰 |
| 成功色 | `#059669` | 翠绿，用于成功状态、通过率 |
| 危险色 | `#e11d48` | 玫瑰红，用于删除、清理操作 |
| 警告色 | `#d97706` | 琥珀，用于禁用切换等 |

## 布局变化

### Header
- 移除现有 `bg-slate-900` 深色背景。
- 改为白色背景（`#ffffff`）+ 底部 1px 边框 `#e8e6e3`。
- 标题字重保持 `font-bold`，副标题颜色改为 `#6b6b6b`。
- 版本号移至更不起眼的位置或使用更浅颜色。

### Tab 导航
- 移除嵌套在深色条内的 Tab 结构。
- 改为 Header 下方独立的导航条，白色背景。
- Tab 按钮默认状态下文字为 `#6b6b6b`，无背景。
- 激活态：文字 `#635bff`，底部 2px 紫色下划线（`border-b-2 border-[#635bff]`）。
- Hover 态：文字 `#1a1a1a`，背景 `bg-[#f5f5f4]`，无圆角或极小圆角。

### 内容区
- 保持 `max-w-6xl mx-auto px-4` 的居中约束。
- 增加上下内边距至 `py-8`（或更大），让页面呼吸感更强。

## 组件规范

### 卡片（渠道列表项、统计卡片）
- 背景：`#ffffff`
- 圆角：`12px`（`rounded-xl`）
- 阴影：`0 1px 3px rgba(0,0,0,0.04), 0 1px 2px rgba(0,0,0,0.02)`（极淡）
- 边框：1px solid `#e8e6e3`（替代现有的 `border-slate-200`）
- Hover 阴影：略微加深至 `0 4px 6px rgba(0,0,0,0.04)`
- 渠道卡片的左侧色条保留，但颜色更新为强调紫 `#635bff`（启用）或 `#e8e6e3`（禁用）。

### 统计卡片（Stats Cards）
- 移除现有的「顶部彩色粗线」设计（`border-t-[3px]`）。
- 改为纯白卡片 + 左侧小色条（4px）或顶部 subtle 渐变线。
- 数字字体更大、字重更粗，标签文字更小、颜色更浅，形成强烈对比。

### 按钮
- **主按钮**（添加、保存）：`bg-[#635bff]` 文字白色，`rounded-xl`，hover `bg-[#554ce6]` + `translateY(-1px)` + 阴影微增。
- **次要按钮**（取消、编辑）：背景透明或 `bg-[#f5f5f4]`，文字 `#1a1a1a` 或 `#635bff`，hover 背景略深。
- **操作按钮**（测试、启用/禁用、删除）：改为更 subtle 的样式，使用文字 + 图标或极小 padding 的 pill 按钮，避免大面积的彩色背景块。例如：
  - 测试：紫色文字 + 淡紫背景 pill
  - 删除：玫瑰色文字 + 淡玫瑰背景 pill
  - 启用/禁用：根据状态使用翠绿/琥珀色文字 + 对应淡色背景 pill

### 表格（API Key 列表、每日趋势）
- 表头：去除 `bg-slate-50`，改为透明背景 + 底部 1px `#e8e6e3` 边框 + 文字 `#6b6b6b` 小写大写字母。
- 行：去除斑马纹，hover 时背景变为 `#faf9f7` 或 `#f5f5f4`。
- 单元格 padding 适当增加，让行高更舒展。

### 标签/徽章（渠道类型、模型名、状态）
- 统一改为 pill 形状（`rounded-full`）。
- 色彩更柔和：
  - 启用：`bg-emerald-50 text-emerald-700`
  - 禁用：`bg-gray-100 text-gray-500`
  - API 类型：`bg-[#f0efff] text-[#635bff]`（淡紫底深紫字）
  - 模型名：`bg-[#f5f5f4] text-[#6b6b6b]`（淡灰底中灰字）

### 模态框
- 遮罩：`bg-black/30` + `backdrop-blur-sm`
- 容器背景：白色
- 圆角：`16px`（`rounded-2xl`）
- 阴影：`0 20px 25px -5px rgba(0,0,0,0.1), 0 8px 10px -6px rgba(0,0,0,0.1)`
- 表单输入框：边框 `#e8e6e3`，focus 时 `ring-[#635bff]`，`rounded-lg`。

### 输入框与表单
- 边框：`1px solid #e8e6e3`
- 圆角：`8px`（`rounded-lg`）
- Focus：`ring-2 ring-[#635bff]/20 border-[#635bff]`
- 背景保持白色。

## 字体与排版
- 字体栈保持 `-apple-system, BlinkMacSystemFont, 'Segoe UI', system-ui, sans-serif`。
- 标题与正文对比更强烈：标签、小标题使用 `text-xs font-medium text-[#6b6b6b] uppercase tracking-wider`。
- 关键数据（统计数字）使用更大的字号和更粗的字重。

## 交互与动画
- 所有颜色、阴影、背景变化增加 `transition-all duration-200 ease-out`。
- 按钮 hover 轻微上移（`-translate-y-0.5`）。
- 卡片 hover 阴影过渡。
- Tab 切换下划线使用 `transition-colors`。

## 功能不变清单
- 所有 JavaScript 逻辑（API 调用、表单提交、Tab 切换、复制、统计渲染）保持原样。
- HTML 结构仅做 CSS class 调整，不增删 DOM 节点。
- Tailwind CSS CDN 引入方式不变。
- 响应式行为不变（当前已有的 `md:grid-cols-2` 等断点保留）。
